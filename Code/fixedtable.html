Demonstração Fixed Table

Fixed Table feita em Angular JS.

A diretiva do Fixed Table necessita que a propriedade "border-collapse" seja "separate". Por esse motivo as bordas ficam dobrados no interior da tabela.
HTML:
<div data-fixed-table="" data-options="options">
    <table class="table table-bordered">
        <thead>
        </thead>
        <tbody>
        </tbody>
    </table>
</div>
JS:
$scope.options = {
    fixedColumns: 2,
    useCustomScroll: true
};
